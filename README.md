## 🚀 Featured Project

### Digital Twin–based Cosmetic Manufacturing Simulator `2026`
*Kiln Visualization Simulator Using Cosmetic Manufacturing Data*

A digital twin solution that makes the **previously unobservable internal mixing states** of stainless steel
reactors visible — enabling AI-based process prediction and real-time monitoring.

**Architecture — Multi-mode simulation**

| Mode | Purpose |
|------|---------|
| **Offline** | User-defined simulation using process variables (mixing ratio, speed, temperature) |
| **Real-time** | Live sensor data integration for monitoring & visualization |
| **Historical** | Replay & analysis of past conditions for root cause analysis |

**Key Capabilities**

- Process variable control: phase properties (oil/water/mixed), temperature, viscosity, input quantity, color
- AI prediction models for process conditions (mixing speed, temperature) and quality outputs (viscosity, pH, mixing state)
- Real-time visualization & monitoring system with sensor data and line-chart tracking

**Impact**

- Improved product quality consistency by reducing variability
- Reduced manufacturing cost and trial experiments
- Shortened production lead time through optimized process conditions
- Accelerated digital transformation toward smart factory implementation
- Serves as a training tool for new operators without raw material loss
